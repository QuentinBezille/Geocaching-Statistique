# Geocaching-Statistique

🧭 Guide d'utilisation : Geocaching Dashboard Pro
Salut ! Voici ton Dashboard de statistiques Geocaching. C'est un outil privé et puissant pour suivre tes trouvailles, tes séries, tes challenges (notamment le 360°) et tes futurs exploits.

🛠️ 1. Installation
Rien à installer ! Il suffit de cliquer sur le lien web que je t'ai envoyé. Le Dashboard s'ouvre directement dans ton navigateur.

⚙️ 2. Configuration Initiale (Indispensable)
Dès ta première ouverture, remplis ces deux champs dans le panneau de contrôle en haut à gauche pour activer les calculs. (Tes réglages s'enregistrent automatiquement sur ton ordinateur pour la prochaine fois !)

Ton Pseudo : Renseigne ton pseudo exact sur Geocaching.com. Cela permet au Dashboard de repérer tes logs personnels dans les fichiers GPX.

Les Coordonnées de ton Domicile : Écris tes coordonnées sous le format classique (ex: N 48° 01.366 E 002° 26.806) ou utilise le bouton 🗺️ pour cliquer directement sur la carte. Ce "point zéro" est vital pour que le Dashboard calcule tes distances et valide ton Challenge 360°.

📥 3. Comment charger tes données
Le Dashboard fusionne tout pour toi :

Fichiers GPX : Glisse tes fichiers .gpx (téléchargés depuis Geocaching.com ou Project-GC) dans la zone dédiée.

Brouillons & Lab Caches : Tu as des brouillons ou des Labs ? Ne t'embête pas à formater le texte ! Copie le texte brut, colle-le dans un fichier Bloc-notes (.txt) et charge ce fichier dans le Dashboard. Le système est intelligent : il nettoie les erreurs de Geocaching.com et fusionne tout automatiquement.

⌨️ 4. Raccourcis et Astuces
Touche P : Active ou désactive le Mode Prévisionnel. Si tu as chargé des brouillons, ce mode simule tes stats comme si tes caches étaient déjà loguées. Parfait pour anticiper tes challenges !

Touche F : Ferme ou rouvre tous les menus du tableau de bord d'un seul coup pour faire place nette sur ton écran.

🔒 5. Confidentialité et Sécurité
100% Local : Cet outil fonctionne entièrement sur ton ordinateur. Aucune donnée n'est envoyée sur internet ni sur un serveur distant. Tes statistiques restent strictement privées.

Nettoyage : Tes fichiers sont stockés dans la mémoire temporaire de ton navigateur. Pour tout remettre à zéro (si tu veux charger des fichiers plus récents), clique simplement sur le bouton rouge "🗑️ Vider les données". Cela effacera tes caches, mais gardera ton Pseudo et ton Domicile en mémoire pour te faire gagner du temps !

Amuse-toi bien avec tes stats ! 🐸🔎
