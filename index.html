🛠️ Comment lire ce calendrier ?

🔍 Recherche Hybride (TXT + GPX)

Le moteur hybride V5 calcule une probabilité de FTF basée sur vos logs et la chronologie.

🛠️ 1. Comment obtenir les 2 fichiers nécessaires ?
  1. 📁 GPX (Zone) : sur Geocaching.com, fais une recherche autour de tes caches récentes (ou une Pocket Query "My Finds" récente), et télécharge le .gpx obtenu. Ce fichier sert à reconstituer la chronologie exacte des logs (qui a loggué avant qui).
  2. 📄 TXT (Project-GC) : sur Project-GC.com, va dans Statistics → Other → My Finds Logs (ou équivalent), filtre sur "FTF" si possible, puis copie-colle le résultat dans un fichier .txt. C'est le fichier "Patron" : il décide quelles caches sont analysées.
  3. Charge les 2 fichiers ci-dessous puis clique sur "Lancer l'analyse 🕵️".
🛠️ 2. Comment transformer un "Oubli" en "Vrai FTF" ?
  1. Clique sur le code GC bleu de la cache pour l'ouvrir sur Geocaching.com.
  2. Modifie ton log "Found it" sur le site.
  3. Ajoute le tag officiel {*FTF*} (ou [FTF]) à l'intérieur de ton texte.
  4. Patiente quelques heures, puis génère un nouveau fichier GPX My Finds (ou exporte à nouveau le TXT de Project-GC).
  5. Importe les nouveaux fichiers ici : le Dashboard basculera automatiquement la cache de l'onglet "Oublis" vers la "🥇 Liste des FTF" !

📊 Habitudes de découverte

🗺️ Cartes des découvertes par région

🛠️ Comment ça marche ?
  1. Cette carte se construit automatiquement à partir du fichier Caches (.gpx) chargé dans la section "1. Données principales" en haut de page.
  2. Pour générer ce GPX : sur Geocaching.com, va dans Mon Profil → Mes Caches Trouvées, puis utilise l'export "My Finds Pocket Query" (ou Project-GC : Tools → My Finds export GPX).
  3. Clique sur un onglet pays pour voir le détail de ce pays, puis clique directement sur une région colorée pour ouvrir le détail des types de caches trouvées dans cette région.
  4. Si une région reste grise alors que tu y as des trouvailles, c'est probablement que le nom de la région dans ton GPX ne correspond à aucun nom connu : vérifie l'orthographe ou signale-le.

🛠️ Créateur de Checker Project-GC & Badge

🛠️ À quoi sert cet outil ?
  1. Il génère le script Lua à coller dans un Checker Project-GC pour créer un défi/challenge personnalisé (ex : "20 caches Earthcache en France avec D≥3").
  2. Remplis les règles à gauche (nombre de caches, mot-clé, tailles, types, zone géo, dates de pose), puis clique sur "Générer le Script Lua".
  3. Copie le résultat généré à droite et colle-le dans la configuration de ton Checker sur Project-GC.com (section Checkers → Create/Edit → Lua Code).
  4. La partie "Badge" génère un petit lien HTML à coller dans la description de ta cache Geocaching.com pour afficher le badge du Checker (nécessite le Code GC et l'ID Checker fournis par Project-GC après création).
